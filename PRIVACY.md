# DialogExport 隐私政策

最后更新：2026-07-17

## 收集和处理的数据

DialogExport 只在用户主动点击“导出当前对话”或“扫描历史会话”后，读取当前支持平台网页 DOM 中用户自己可见的内容，包括对话文本、标题和来源链接。扫描 ChatGPT 历史会话时，为识别尚未展开的项目，扩展会临时取得当前 ChatGPT 登录会话的访问凭据，并仅向 `chatgpt.com` 自身的项目接口请求项目会话的标题和 ID；访问凭据只保存在内存中，不会存储或发送给其他服务。扩展不读取密码、IndexedDB、无关的网站网络请求或浏览器历史。

## 数据用途和传输

读取的内容只用于完成用户主动请求的本地扫描与导出。除上述由用户主动触发的 ChatGPT 同源项目列表请求外，扩展没有开发者服务器、统计服务、广告 SDK 或远程配置，不会把聊天内容、访问凭据、URL 或错误报告发送给开发者或第三方。

## 本地存储

Popup 的扫描列表、选择项和导出状态可能临时保存到 `chrome.storage.session`，用于 Popup 关闭后重新打开时恢复界面。数据按标签页和网站来源隔离，浏览器会话结束后由浏览器清理。导出的文件只由浏览器下载 API 保存到用户选择的下载位置。

## 用户控制

用户可以随时卸载扩展，或在浏览器扩展详情页清除扩展数据。扩展不会在用户没有点击操作时读取支持网站的对话内容。

## 联系方式

如需报告隐私或安全问题，请通过项目仓库的 Issue/联系方式联系维护者。
